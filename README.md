## ✨ Features

| Module | Description |
|---|---|
| 🏠 Dashboard | Balance, spending stats, charts, savings overview |
| ➕ Add Expense | Quick entry with category auto-detect & recurring support |
| 📋 History | Searchable, filterable transaction log |
| 🎯 Budget Planner | Per-category budgets with alerts & smart suggestions |
| 💰 Savings Goals | Goal cards, progress bars, quick deposits |
| 📊 Analytics | Monthly trends, category breakdown, AI insights |
| 🏅 Achievements | Gamified badges to reward good habits |
| 💬 Feedback | Rating system, bug reports, FAQ, financial tips |

## 🚀 Getting Started

### Prerequisites
- Node.js ≥ 16
- npm or yarn

### Install & Run

```bash
# Clone the repo
git clone https://github.com/dslashhonyebadgerr77-debug/personal-finance-tracker.git
cd personal-finance-tracker

# Install dependencies
npm install

# Start development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

## 🎨 Tech Stack

- **Frontend:** React 18 + CSS Variables (no external UI library)
- **Charts:** Custom SVG (Pie, Bar, Sparkline — zero dependencies)
- **Fonts:** Cabinet Grotesk + DM Sans (Google Fonts)
- **State:** React useState (local, no Redux needed)
- **Storage:** In-memory (localStorage integration ready)

---

## 🧩 Design Philosophy

Built using **Design Thinking**:
1. **Empathize** — Designed for real pain points: forgetting to track, confusing UIs
2. **Define** — Core problem: complexity kills habit formation
3. **Ideate** — One-tap entry, visual feedback, gamification
4. **Prototype** — All key screens: Dashboard, Add, Budget, Savings
5. **Test** — Feedback form, ratings, error validation
